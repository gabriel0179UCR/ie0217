Todos los archivos se compilar y ejecutar de la siguiente forma:

Compilar:
```
g++ -o NOMBRE_ARCHIVO.exe .\NOMBRE_ARCHIVO.cpp
```
Ejecutar:
```
.NOMBRE_ARCHIVO.exe
```
