## Clase 7: C++ Biblioteca STL

Todos los archivos se compilan y ejecutan de la siguiente forma:

Compilar:
```
g++ -o NOMBRE_ARCHIVO.exe .\NOMBRE_ARCHIVO.cpp
```
Ejecutar:
```
.\NOMBRE_ARCHIVO.exe
```
